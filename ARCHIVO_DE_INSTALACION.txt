🚀 INSTALACIÓN RÁPIDA:

1. Ir al directorio del frontend:
   cd frontend

2. Instalar dependencias (si no lo has hecho):
   npm install

3. Iniciar servidor de desarrollo:
   npm run dev

4. Abrir en el navegador:
   http://localhost:5173

✅ ¡LISTO! Verás tu hermoso Homepage PESV Digital
